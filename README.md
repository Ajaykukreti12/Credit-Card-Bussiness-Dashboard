#💳 Credit Card Business Analytics Dashboard (Excel)

"📊 Project Overview"

This project presents an interactive Credit Card Business Dashboard built in Microsoft Excel to analyze transaction performance, customer behavior, and card usage patterns.

The dashboard transforms raw transactional data into actionable business insights using Excel features like Power Query, Data Modeling, Pivot Tables, and Interactive Charts.

The goal of this project is to simulate how financial institutions analyze credit card transactions to understand customer spending behavior, revenue contribution, and credit risk distribution.



"🎯 Business Objectives"

The dashboard answers key business questions such as:

How much total revenue is generated from credit card transactions?

Which card brands generate the highest revenue?

What are the most common transaction types?

Which cities generate the highest transaction volume?

How does spending differ by age group and gender?

What percentage of customers have high credit scores?

Which cards have the highest transaction value?

How many cards are expiring in upcoming years?


"📂 Dataset Structure"

The dataset consists of three different tables:

'1️⃣ Users Data'

Contains customer demographic information.

Fields include:

Client ID

Age

Gender

Address

Income

Credit Score

Number of Credit Cards


'2️⃣ Card Data'

Contains credit card information.

Fields include:

Card ID

Card Brand

Card Type

Expiry Date

Credit Limit

Chip Availability


'3️⃣ Transaction Data'

Contains transaction level details.

Fields include:

Transaction ID

Client ID

Card ID

Transaction Amount

Merchant City

Merchant State

Transaction Type

Payment Method (Chip / Swipe)

"⚙️ Data Preparation Process"

The raw dataset was processed using Excel Power Query:


"Steps performed:"

Imported three separate datasets into Excel

Cleaned and transformed the data

Merged tables using Client ID and Card ID

Created a Master Data Table

Removed duplicate columns

Built a data model for analysis

"📊 Key Insights"

Some important insights from the analysis:

Swipe transactions dominate payments (~86%), indicating customers prefer physical card usage.

Visa and Mastercard generate the majority of revenue, showing strong market dominance.

Senior customers contribute the highest spending volume.

65% of customers have high credit scores, indicating a relatively low credit risk portfolio.

Certain cities contribute significantly higher transaction volumes compared to others.
